# Wireless Relay Control using Bluetooth
The code is written using Arduino IDE 1.8.9 software
The code is then uploaded into the Arduino UNO Board
The data is read using Bluetooth Terminal HC-05 application

Reads the data from Bluetooth module connected to the Arduino via Bluetooth terminal HC 05 application to trigger the relay which is also connected to the Arduino that acts as a
switch to light the bulb on or off externally.
